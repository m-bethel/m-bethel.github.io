# Sovereign Citizens Party Website

A professional political party website created for an educational assignment, showcasing a hypothetical political party platform and values.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Aesthetics**: Clean, editorial-style design with smooth animations
- **Sections Include**:
  - Hero section with party introduction
  - About section explaining the party's mission
  - Platform section with 6 core policy areas
  - Values section highlighting key principles
  - Contact/Join section with form

## Files Included

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and animations
- `script.js` - Interactive features and form handling

## Deploying to GitHub Pages

### Method 1: Using GitHub Web Interface

1. **Create a new repository**:
   - Go to [GitHub](https://github.com) and sign in
   - Click the "+" icon (top right) → "New repository"
   - Name it `yourusername.github.io` (replace `yourusername` with your actual GitHub username)
   - Keep it Public
   - Click "Create repository"

2. **Upload files**:
   - On your new repository page, click "uploading an existing file"
   - Drag and drop all three files: `index.html`, `styles.css`, and `script.js`
   - Scroll down and click "Commit changes"

3. **Enable GitHub Pages** (if not auto-enabled):
   - Go to Settings → Pages (left sidebar)
   - Under "Source", select "main" branch
   - Click Save

4. **Access your site**:
   - Your site will be live at: `https://yourusername.github.io`
   - It may take a few minutes to deploy

### Method 2: Using Git Command Line

If you're familiar with Git:

```bash
# Clone your repository
git clone https://github.com/yourusername/yourusername.github.io.git
cd yourusername.github.io

# Copy the website files to this directory
# (copy index.html, styles.css, script.js)

# Add and commit files
git add .
git commit -m "Add Sovereign Citizens Party website"

# Push to GitHub
git push origin main
```

Your site will be live at `https://yourusername.github.io`

## Alternative: Project Repository

If you want to use a different repository name:

1. Create a repository with any name (e.g., `political-party-website`)
2. Upload the files
3. Go to Settings → Pages
4. Select "main" branch as source
5. Your site will be at: `https://yourusername.github.io/political-party-website`

## Customization Tips

- **Colors**: Edit CSS variables in `styles.css` (lines 1-10)
- **Content**: Modify text in `index.html`
- **Fonts**: Change Google Fonts in the `<head>` section
- **Platform Points**: Add/remove platform cards in the Platform section
